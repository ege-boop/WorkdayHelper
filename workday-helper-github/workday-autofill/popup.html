Workday Helper

Many respected companies use Workday. This tool helps you fill out most parts of the form automatically so you can complete more applications faster.
Note: This tool fills common questions only. You'll still need to answer company-specific questions manually.
What gets filled:
• Personal info (name, address, phone)
• Common questions (visa, salary, education)
• Demographics (if requested)

Works on: *.myworkdayjobs.com
